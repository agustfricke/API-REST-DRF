## COMO USAR ==>
- git clone https://github.com/agustfricke/API-REST-DRF
- cd API-REST-DRF
- python3 -m venv venv
- source venv/bin/activate
- pip install - requirements.txt
- cd crud
- python3 manage.py runserver
